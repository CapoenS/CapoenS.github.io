# Personal page assets

Files used by `me.html`:

- **Profile photo** → `assets/images/me/profile.jpg`
  - Square-ish image works best (it's shown in a circle).
  - If the file is missing, the page shows an "SC" initials placeholder instead — no error.

- **Résumé** → `assets/resume.pdf`
  - The "Download résumé (PDF)" button links here.
  - Until you add the file, clicking the button just 404s — drop your PDF at that path
    (project root `assets/` folder) to enable it.

Lowercase filenames, no spaces (GitHub Pages is case-sensitive).
